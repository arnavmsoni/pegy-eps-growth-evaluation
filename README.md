# PEGY EPS Growth Source Evaluation

This project evaluates how different analyst-provided EPS growth estimates impact the PEGY ratio’s ability to identify undervalued stocks.

## Data Sources
- Alpha Vantage (earnings and EPS estimates)
- Financial Modeling Prep (EPS estimates, P/E, dividend yield, prices)
- Twelve Data (earnings and EPS estimates)

## Goal
Compare PEGY variants constructed using different EPS growth sources and evaluate their screening performance using forward excess returns.

Code and experiments will be developed in later milestones.
